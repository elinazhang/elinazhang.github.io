  1. Sign up for GitHub
    • A GitHub account is needed to use GitHub Pages feature. Sign up here if you don’t have one.
    • If it’s the first time you use it, I recommend the Hello World guide to help get familiar with the git workflow.
    • BTW, free users can only set up public repositories.
