Setting up a GitHub repository for GitHub Pages

  1. Sign up for GitHub
    • A GitHub account is needed to use GitHub Pages feature. Sign up here if you don’t have one.
    • If it’s the first time you use it, I recommend the Hello World guide to help get familiar with the git workflow.
    • BTW, free users can only set up public repositories.

  2. Create a SSH Key for your account

    • Open Terminal, then paste the text below, with your_email@example.com replaced with your GitHub email address.

    • Press Enter when you’re prompted to enter location and passphrase, which will then use default settings.

    • By default, you can find file id_rsa.pub under /var/{userName}/.ssh/. Copy the ssh key it contains.
    • Go back to github -> Settings -> SSH and GPG keys -> New SSH Key -> Paste and Add ( shortcut )
